Shared Folder

The MHTTC Shared Folder contains many important and valuable resources for the Network, such as:

  • GPRA and Evaluation documents
  • Working Group agendas, notes, and recordings
    • Cultural Responsiveness Working Group
    • Dissemination & Implementation Working Group
    • Early Psychosis Working Group
    • Workforce Development Working Group
  • Instructional Design and Course Development on HealtheKnowledge
  • Branding materials, including MHTTC logos, Center images, website maintenance tips, and more
  • School Mental Health Initiative information
  • MHTTC Network Coordinating Office Trainings
  • MHTTC Reports
  • Notes from previous MHTTC Meetings
  • MHTTC Staff Roster
  • MHTTC and TTC Guidance Documents
  • And more!

There is also a recorded MHTTC Orientation for new staff under: \MHTTC or TTC Operating Procedures\MHTTC NCO Trainings.

Quick Links

MHTTC Shared Folder

This is the main folder available to all MHTTC Staff, containing information on working groups, GPRA, the TTC Hub, materials for new staff members, and more! Links 2-9 are located within this main folder.

MHTTC Style Guide

This guide contains information on how to brand materials provided by your Center, as well as information on the various colors, fonts, and other elements used by the Network.

MHTTC New Staff Orientation

This orientation is for new staff members to learn more about the MHTTC Network and what we do.

MHTTC Staff List

This list contains the names and emails for all MHTTC Centers across the country.

MHTTC and Other Logos

This folder contains logos that Centers can utilize for marketing purposes. They are available in black, white, and gray and orange.

MHTTC Website Resources

MHTTC and TTC Groups

This folder contains information on the 4 Working Groups and 5 Coordination Groups within the MHTTC Network, and the 6 groups across the TTC Network that the MHTTC Network is a part of.

GPRA Documents/Resources

This folder contains information on how to utilize GPRA for your events, including generating event codes and creating language for post-event and follow-up surveys.

GPRA Online

This link is to the GPRA system, operated by Lanitek, where you can create events and send post-event and follow-up surveys for evaluation purposes.

SPARS Data Download and Visualization

This link is to the SPARS database, where you can download and visualize data from all events that you have GPRAd.
